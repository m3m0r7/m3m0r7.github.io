Fill the function name in PHP

About this game

  • This game is racing to fill the empty input in PHP functions. Please careful at disables an IME when playing.
  • For example, please enter "reduce" or "filter" when showing "array_[_][_][_][_][_][_]".
  • Move to next question when your input is correcting but if incorrect your input, erasing field and restarted to input.
  • Each questions are having limited times(Easy=16sec, Hard=8sec, Crazy=4sec), and decreasing this time on this game play by play without easy mode.
  • Each questions are having limited times（Easy=fixed 16 sec, Hard=decreasing from 16 sec, Crazy=decreasing from 8 sec）。
  • Move to next input when you input a character, and to empty previous input when entering a Backspace key or Delete key.
  • Your score increase by chosen more hard mode and going to next game. Just share your score on Twitter!
